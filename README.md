# Zotero PDF Fulltext Search

一个适用于 Zotero 6 的 PDF 全文检索插件。

## 功能

- 在 Zotero 主界面添加 PDF 全文检索框
- 支持检索 Zotero 已索引的 PDF 正文内容
- 在当前界面直接展示 PDF 命中片段
- 支持关键词高亮
- 支持上一个 / 下一个结果跳转
- 支持定位条目、打开 PDF、复制片段

## 适用版本

- Zotero 6.x

暂不支持 Zotero 7。

## 安装方法

1. 下载 `.xpi` 文件
2. 打开 Zotero
3. 进入 `Tools` → `Add-ons`
4. 点击齿轮图标
5. 选择 `Install Add-on From File...`
6. 选择 `.xpi` 文件
7. 重启 Zotero

## 注意事项

本插件依赖 Zotero 自身的 PDF 全文索引功能。

如果 PDF 是扫描版，或者 Zotero 尚未建立全文索引，可能无法检索到正文内容。  
可以尝试在 Zotero 中右键 PDF，选择重新索引。
