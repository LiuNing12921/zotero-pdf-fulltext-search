# Zotero PDF Fulltext Search v0.3.5 (Zotero 6)

作者：Loge

## 本版修改

- 重新调整顶部工具栏布局：搜索框、检索按钮、清空按钮统一为紧凑高度，并整体垂直居中。
- 搜索框高度进一步压低，避免上一版显得过高。
- 对“检索 / 清空”按钮增加专门样式，减少按钮文字偏上或偏下的问题。
- 保留“上一个 / 下一个”按钮在结果摘要行右侧的位置。
- 搜索结果仍直接显示在当前主界面中，不再弹出独立窗口。
- PDF 正文命中词继续高亮展示。

## 使用

在 Zotero 主界面顶部、原生搜索框左侧的“PDF全文”输入框中输入关键词或短语，按 Enter 或点击“检索”。

结果会显示在中间条目列表上方，每个 PDF 默认展示最多 4 段上下文片段，并高亮命中词。点击结果摘要行右侧的“上一个 / 下一个”可以在命中的 PDF 结果之间切换。

## 说明

本插件依赖 Zotero 已建立的 PDF 全文索引。如果 PDF 为扫描件或未建立全文缓存，请先 OCR，或在 Zotero 中对附件执行 Reindex Item。
